# C++ Learning Journey 🚀

Welcome to my C++ learning repository! This project documents my progress learning C++ from the very basics to more advanced concepts.

## 📚 Folder Structure

Each folder represents a major topic in C++ learning, organized progressively:

1. **01-basics** - Hello World, variables, data types, I/O
2. **02-arithmetic** - Arithmetic operators and operations
3. **03-conditionals** - If/else statements and logical operators
4. **04-loops** - For, while, do-while loops
5. **05-functions** - Function declarations, parameters, recursion
6. **06-arrays** - Arrays, multidimensional arrays, strings
7. **07-pointers** - Pointers, memory addresses, pointer arithmetic
8. **08-classes-objects** - OOP basics, classes, objects
9. **09-inheritance** - Inheritance, virtual functions, polymorphism
10. **10-file-io** - File input/output operations

## 🎯 How to Use

- Each folder has a README with specific learning objectives
- Add your code files (`.cpp`) to each folder as you learn
- Check off items in each README as you complete them
- Practice with examples and exercises for each topic
